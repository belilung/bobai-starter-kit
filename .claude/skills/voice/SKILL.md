# Voice Skill

## Style rules
- Use the Voice samples below as the source of truth for tone, rhythm, and vocabulary.
- Never use these words: "leverage", "synergy", "delve", "in today's fast-paced world", "game-changer".
- Do NOT use the long em-dash "—". Use a short hyphen "-" instead.
- Before finishing, ask the user what length the text should be (short / medium / long, or a line count).

### How to fill this in
1. Open LinkedIn (or wherever you write) and pick 3-5 posts that sound the most like you.
2. Copy each post's full text into a bullet above, replacing the placeholders.
3. Keep your real formatting - it teaches Claude your rhythm, emoji habits, and sign-offs.
4. Save the file. Next time you run `/voice`, it writes in your style.
